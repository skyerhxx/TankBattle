# TankBattle

**坦克大战游戏开发项目(Python)**——**基于pygame**

</br>

一个简易版的坦克大战游戏，基本功能都实现了，也有添加音效

</br>

**开发环境：**

Python：3.7

pygame: 1.9.6

IDE: vscode

</br>

程序主体：

```
BaseItem
MainGame
Tank
MyTank
EnemyTank
Bullet
Wall
Explode
Music
```
